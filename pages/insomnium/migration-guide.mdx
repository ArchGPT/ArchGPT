<br/>
<b>There are two ways to migrate data from Insomnia to Insomnium.</b>

1. In Insomnia, use the GUI button <kbd >Export files from the "Insomnia" Project</kbd>  under `Perference/Data` to export files (e.g. to your `~/Downloads`). <br/><br/>After that, open Insomnium and import these files using the <kbd >Import to the "Insomnium" Project</kbd> button under its `Perference/Data`.

2. Use the terminal directly. 

Linux:

```
cp -r ~/.config/Insomnia ~/.config/Insomnium
# or 
cp -r $XDG_CONFIG_HOME/Insomnia $XDG_CONFIG_HOME/Insomnium
```

MacOS:

```
cp -r ~/Library/Application\ Support/Insomnia/*.db ~/Library/Application\ Support/Insomnium/*.db
```

Windows:

```
cp -r "%APPDATA%\Roaming\Insomnia" "%APPDATA%\Roaming\Insomnium"
```
